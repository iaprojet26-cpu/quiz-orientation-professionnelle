# Articles SEO - Quiz Orientation

Ce dossier contient 30 articles optimisés SEO en 3 langues (français, anglais, arabe).

## Structure

Chaque article est dans un dossier `article-XX/` contenant :
- `fr.md` - Version française (900-1400 mots)
- `en.md` - Version anglaise (900-1400 mots)
- `ar.md` - Version arabe (900-1400 mots)
- `metadata.json` - Métadonnées multilingues
- `schema.json` - JSON-LD Article schema
