# Quelles compétences mettre sur un CV en 2024

En 2024, le marché du travail évolue rapidement et les compétences recherchées par les employeurs changent. Savoir quelles compétences mettre sur votre CV et comment les présenter peut faire la différence entre une candidature retenue et une candidature ignorée. Voici un guide complet pour optimiser la section compétences de votre CV.

## Pourquoi la section compétences est cruciale

La section compétences est l'une des premières sections que les recruteurs consultent. Elle permet de :

- **Identifier rapidement votre profil** : Les recruteurs scannent votre CV en 6 secondes
- **Passer les filtres ATS** : Les logiciels de recrutement recherchent des mots-clés spécifiques
- **Démontrer votre valeur** : Montrer ce que vous pouvez apporter à l'entreprise
- **Vous différencier** : Mettre en avant vos atouts uniques

## Les compétences techniques (Hard Skills)

### Compétences informatiques essentielles en 2024

#### Développement Web
- **Front-end** : React, Vue.js, Angular, HTML5, CSS3, JavaScript/TypeScript
- **Back-end** : Node.js, Python (Django, Flask), Java (Spring), PHP (Laravel)
- **Bases de données** : MySQL, PostgreSQL, MongoDB, Redis
- **Outils** : Git, Docker, CI/CD, AWS, Azure

#### Data & Analytics
- **Analyse de données** : Python (Pandas, NumPy), R, SQL
- **Visualisation** : Tableau, Power BI, D3.js
- **Machine Learning** : TensorFlow, PyTorch, Scikit-learn

#### Design & Créativité
- **Design graphique** : Adobe Photoshop, Illustrator, InDesign, Figma
- **UI/UX Design** : Wireframing, Prototypage, Design Thinking
- **Montage vidéo** : Premiere Pro, After Effects, Final Cut Pro

### Compétences métier par secteur

#### Marketing Digital
- SEO/SEM, Google Analytics, Google Ads, Facebook Ads
- Content Marketing, Email Marketing, Marketing Automation
- Réseaux sociaux (LinkedIn, Instagram, TikTok)

#### Finance & Comptabilité
- Comptabilité générale, Gestion financière, Analyse financière

#### Ressources Humaines
- Recrutement, Gestion de carrière, Formation

- Droit du travail, GPEC

## Les compétences comportementales (Soft Skills)

Les soft skills sont de plus en plus valorisées en 2024. Voici les plus recherchées :

### Communication
- **Communication écrite** : Rédaction claire, présentation professionnelle
- **Communication orale** : Présentation, négociation, écoute active
- **Communication interculturelle** : Travail dans des équipes internationales

### Leadership & Management
- **Gestion d'équipe** : Animation, motivation, délégation
- **Prise de décision** : Analyse, résolution de problèmes complexes
- **Gestion de projet** : Planification, organisation, suivi

### Adaptabilité & Apprentissage
- **Agilité** : Capacité à s'adapter rapidement aux changements
- **Apprentissage continu** : Curiosité, autoformation, veille technologique
- **Résilience** : Gestion du stress, capacité à rebondir

### Collaboration
- **Travail d'équipe** : Collaboration efficace, esprit d'équipe
- **Intelligence émotionnelle** : Empathie, gestion des relations
- **Résolution de conflits** : Médiation, négociation

## Comment présenter vos compétences sur le CV

### 1. Organisez par catégories

Ne listez pas toutes vos compétences en vrac. Organisez-les :

```
COMPÉTENCES TECHNIQUES
- Langages : JavaScript, Python, Java
- Frameworks : React, Node.js, Spring Boot
- Outils : Git, Docker, AWS

COMPÉTENCES LINGUISTIQUES
- Arabe : Langue maternelle
- Français : Professionnel
- Anglais : C1 (TOEIC 850)

COMPÉTENCES TRANSVERSALES
- Gestion de projet, Leadership, Communication
```

### 2. Utilisez des niveaux de maîtrise

Indiquez votre niveau pour chaque compétence :

- **Expert** : Maîtrise complète, plusieurs années d'expérience
- **Avancé** : Bonne maîtrise, utilisation régulière
- **Intermédiaire** : Connaissances solides, utilisation occasionnelle
- **Débutant** : Notions de base, en apprentissage

**Exemple :**
- React : Expert (5 ans d'expérience)
- Vue.js : Avancé (2 ans d'expérience)
- Angular : Intermédiaire (projets personnels)

### 3. Adaptez aux offres d'emploi

Analysez l'offre et mettez en avant les compétences les plus pertinentes :

- **Lisez attentivement l'offre** : Identifiez les compétences clés mentionnées
- **Priorisez** : Mettez en premier les compétences demandées
- **Utilisez les mêmes termes** : Reprenez les mots-clés de l'offre

### 4. Quantifiez quand c'est possible

Ajoutez des preuves concrètes :

- "Gestion de projet : 10+ projets menés à terme"
- "Python : 3 ans d'expérience, certification Google"
- "Anglais : TOEIC 920, expérience internationale (6 mois)"

## Erreurs à éviter

### ❌ Liste trop longue

Ne listez pas 50 compétences. Sélectionnez les 10-15 les plus pertinentes.

### ❌ Compétences non vérifiables

Évitez les compétences trop vagues comme "Créativité" sans contexte.

### ❌ Mensonges

Ne mentionnez pas des compétences que vous ne maîtrisez pas. Vous serez testé en entretien.

### ❌ Compétences obsolètes

Mettez à jour régulièrement. Certaines technologies deviennent obsolètes.
